# Cornelius Aurelius Beta Reader

Official static GitHub Pages website for Cornelius Aurelius's free beta-reading service.

## Custom domain
https://beta-reader.corneliusaurelius.com/

## Upload
Upload every file in this folder to the root of the `cornelius-aurelius-beta-reader` repository, enable GitHub Pages from the `main` branch `/root`, and set the custom domain to `beta-reader.corneliusaurelius.com`.
